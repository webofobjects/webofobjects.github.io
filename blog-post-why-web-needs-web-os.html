Why the Web Needs a Web Operating System

Essay January 2025 by Valentin — Web Of Objects

The web we use today was never designed for how we think. Files, folders, pages, and feeds were created for a simpler era — an era before AI agents, real-time collaboration, and global-scale collective knowledge.

Every app invents its own data model, its own formats, and its own walled garden. Your ideas are scattered across notes apps, docs, chats, task managers, and proprietary databases — none of which truly talk to each other. The more tools you use, the more fragmented your mind becomes.

Beyond files and pages

Files and pages are containers. They were useful when computing was mostly about storing documents. But today, knowledge is dynamic, relational, and alive. A single idea can be:

  • a paragraph in a document,
  • a bullet in a meeting note,
  • a task in a project board,
  • a message in a chat,
  • a row in a spreadsheet.

These fragments are logically connected — but the web doesn’t understand that. As humans, we carry the burden of remembering how everything fits together.

The case for a Web Operating System

A Web Operating System treats ideas not as files, but as objects in a shared universe. An object could represent:

  • a person,
  • a concept,
  • a project,
  • an experiment,
  • a decision.

Each object can hold structure, unstructured notes, relationships, history, and behavior. Instead of asking “Where did I write that?”, you ask “Which objects are related to this one?” — and the system can answer.

This is what Web Of Objects aims to build: a universal layer for representing and connecting information so that humans, AI agents, and applications can all operate in the same space.

Technology is not enough

But a Web Operating System alone is not the end goal. Technology can amplify whatever system we plug it into — including systems that leave most people behind.

That’s why Web Of Objects is not just a technical project; it’s a call to invent a Web Society — a way of organizing technology, business, and cooperation where progress is shared, not hoarded.

This blog will document that journey: the technical foundations of Web Of Objects, the social and economic questions it raises, and the practical steps we’re taking to turn a vision into a working prototype and a global movement.

If this resonates with you, you’re early. And you’re invited.
